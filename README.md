# palavra

Palavra is a Node.js module that generates one or more random portuguese words.

## Installation

	$ npm install [-g] palavra

## Usage

```javascript

> const palavra = require("palavra")

> palavra()
'parelha'

> palavra(3)
'adverbial documentacao longe'

```
